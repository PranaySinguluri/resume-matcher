How to get a Gemini API key

  1. Go to aistudio.google.com/apikey
  2. Sign in with any Google account
  3. Click Create API key
  4. Copy the key (starts with AIza...)
  5. Paste it into the API Key field on this page
Free tier: 1500 requests per day on Gemini 2.5 Flash. No credit card required.
Privacy: The key is used only for this browser session and is never saved. It goes directly from your browser to Google's API, never through any other server.
